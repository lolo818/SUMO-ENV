# sumo如何運作

## 首先需要三個最主要的檔案

* .net.xml: 用來定義道路

* .rou.xml: 用來定義車流量

* .sumocfg: 用來把上面兩個檔案結合形成一個專案讓sumo執行

## 如何產生.net.xml和.rou.xml檔

* .net.xml: 用netedit gui畫出來

* .rou.xml: 使用.trip.xml 配合duarouter產生
